Трассировка автономных систем
Версия 1.0 (13.04.2019)

Автор: Муравейник Алексей (alexmur20000@mail.ru)


Описание
    Данное приложение позволяет получить информацию о маршрутизаторах (ip адрес, номер автономной системы),
    через которые проходит пакет, до указанного узлав удобном для восприятия виде.
    Для поиска ip адресов маршрутизаторов используется утилита "tracert"
    Для получения номера автономных систем используется whois сервис "api.whois.vu"


Требования
    Python версии 3


Состав
    1. Программа для сервера (компютера, отвечающего за распределение информации): server.py
    2. Программа для узла (компьютера, на котором будет храниться информация): client.py
        2.1 Класс, инкапсулирующий работу с хранением данных - storage.py
        2.2 Класс, содержащий логику префиксного дерева - trie.py
    3. Программа для конечного пользователя: requester.py


Использование приложения

    Справка по запуску:
            python as_route.py
            python as_route.py -h
            python as_route.py --help


    В качестве единственного аргумента командной строки следует указать ip адрес конечного узла, либо его доменное имя
            python as_route.py vk.com
            python as_route.py 87.240.190.78
