Трассировка автономных систем
Версия 1.0 (13.04.2020)

Автор: Муравейник Алексей (alexmur20000@mail.ru)


Описание
    Данное приложение позволяет получить информацию о маршрутизаторах (ip адрес, номер автономной системы),
    через которые проходит пакет, до указанного узла в удобном для восприятия виде.
    Для поиска ip адресов маршрутизаторов используется утилита "tracert"
    Для получения номера автономных систем используется утилита "curl"
    с обращением к whois сервису "api.whois.vu"


Требования
    Python версии 3


Использование приложения

    Справка по запуску:
            python as_route.py
            python as_route.py -h
            python as_route.py --help


    В качестве единственного аргумента командной строки следует указать ip адрес конечного узла, либо его доменное имя
            python as_route.py vk.com
            python as_route.py 87.240.190.78
